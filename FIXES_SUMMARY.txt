================================================================================
HOW IT WORKS NOW
================================================================================

Detection Pipeline:
  1. File Input → Validate (exists, <100MB)
  2. Parse PE Header → Extract features using pefile
  3. Build Feature Vector → 16 EMBER-compatible features
  4. Model Prediction → EMBER RF gives malware probability
  5. Threshold Check → If prob > 0.50 → MALICIOUS
  6. Output Result → Display verdict + confidence + hashes

Features Extracted:
  - file_size (raw byte count)
  - has_signature (PE authenticated?)
  - machine_type_*[11] (one-hot: I386, AMD64, ARM, ARMNT, IA64, etc.)
  - characteristics_count (PE characteristics flags)
  - histogram_sum (byte histogram sum)
  - entropy_mean (Shannon entropy)
